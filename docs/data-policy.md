# Data Policy

This project is designed for public portfolio use. It must never depend on confidential workplace material.

## Allowed Data

- Public log datasets such as Loghub.
- Public incident reports and postmortems.
- Public technical documentation.
- Synthetic debug cases.
- Logs generated by local demo applications.

## Forbidden Data

- Employer or client logs.
- Private stack traces.
- Real customer records.
- Internal runbooks or architecture documents.
- Secrets, tokens, private hostnames, or credentials.

## Review Rule

Every seeded case must say whether it is synthetic, public, or local-demo generated. Synthetic data should be realistic enough to teach engineering concepts but fictional enough to be safe.
