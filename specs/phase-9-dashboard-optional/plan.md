# Phase 9 Plan: Optional Dashboard

## Approach

Keep this phase deliberately optional. Use it to make the platform easier to show, not to compensate for missing backend, evaluation, or DevOps maturity.

## Candidate Views

- Query demo.
- Evaluation status.
- Ingestion status.
- Health and readiness status.
- Basic observability summary.

## Validation

- Dashboard uses live API data.
- Screenshots are current.
- Demo script remains accurate.
