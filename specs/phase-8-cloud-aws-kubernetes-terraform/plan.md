# Phase 8 Plan: Cloud / AWS / Kubernetes / Terraform

## Approach

Start from the validated containerized platform. Decide the lowest-cost AWS path before provisioning. Prefer local Kubernetes first if it improves learning without creating unnecessary cloud spend.

## Target Capabilities

- Terraform provisioning.
- Image publishing.
- Deployment workflow.
- Secrets and environment configuration.
- Logs and metrics after deployment.

## Validation

- Terraform plan and apply workflow.
- Deployment from built image.
- Operational smoke test.
- Cost and teardown documentation.
